# SportX — Private Performance Landing Page

**Live Demo:** https://landing-page-sportx.vercel.app/


![SportX Preview](<img width="1897" height="861" alt="sportX" src="https://github.com/user-attachments/assets/677f9c2b-2e2c-46d6-991b-9df694b33064" />)

Премиальный одностраничный лендинг для сервиса персональных тренировок и performance-коучинга.  
Без сборщиков и фреймворков.

---

## Стек
- HTML5 (семантика)
- CSS3 (CSS Variables, Grid, Flexbox, адаптив)
- Vanilla JavaScript (ES6)
- Google Fonts
- IntersectionObserver API

---

## Функциональность
- Плавная навигация по якорям
- Reveal-анимации при скролле
- Адаптивная верстка (desktop / tablet / mobile)
- Форма лидогенерации с клиентской валидацией
- Поддержка `prefers-reduced-motion`
- Четкая визуальная иерархия

---

## Запуск
1. Клонировать репозиторий  
2. Открыть `index.html` в браузере

---

## JavaScript
- Submit-логика формы (валидация, reset)
- Smooth scroll по якорям
- Scroll-анимации через `IntersectionObserver`
- Управление задержками через `data-delay`

---

## CSS
- Единая дизайн-система через `:root`
- Компонентный подход
- Media queries для адаптива
- Анимации без библиотек

---

## Ограничения
- Нет backend
- Отправка формы — заглушка
- Нет SSR и SEO-оптимизации
- Нет сборки и минификации

---

## Назначение
- Портфолио frontend-разработчика
- Демонстрация premium UI/UX
- База для коммерческого лендинга

---

## Лицензия
Только демонстрационные и учебные цели.
